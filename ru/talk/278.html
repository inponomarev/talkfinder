UI Automator deep diving

Светлана Смельчакова

Доклад о том, как работает UI Automator и Accessibility Service.

UI Automator — инструмент Android SDK для автоматизированного UI-тестирования. Светлана подробно рассмотрит все возможности и неочевидные особенности API, как устроен UI Automator, а также какие есть проблемы и ограничения в этом инструменте.

Комментарий программного комитета:

Обычно рассказ об автотестах сводится к обзору подходов и фреймворков. Но не в этот раз! Светлана расскажет о внутренностях UI Automator — фреймворка, который используется для тестирования приложений методом черного ящика. Вы узнаете, почему автотесты ведут себя именно так, какое API и системные компоненты используются, чем фреймворк отличается от Espresso, и просто узнаете про кишки Android.
