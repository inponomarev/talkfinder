Kotlin как язык для разработки фронтенда

Залим Башоров

Взаимодействие с JS-кодом, отличия от Kotlin для JVM и не только.

Kotlin — относительно новый язык программирования, который изначально разрабатывался для JVM, но сегодня может компилироваться в JavaScript и не только.

В данном докладе будет рассказано про различные аспекты Kotlin JS:

  • взаимодействие с кодом на JavaScript;

  • какие есть отличия от Kotlin на JVM;

  • как использовать один и тот же код для разных платформ;

  • и т.д.
