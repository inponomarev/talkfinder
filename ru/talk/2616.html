Воркшоп: Стартуем в тестировании производительности

Сергей Махетов

Воркшоп для тестировщиков, которые хотят заняться тестированием производительности, но не знают, с чего начать.

<p>Воркшоп для тестировщиков, которые хотят заняться тестированием производительности, но не знают, с чего начать.</p> <p>Создадим тест с нуля на Jmeter. Пошагово сделаем базовый запрос, рассмотрим способы задания профиля нагрузки, метрики тестирования, предоставление отчетов (консоль, графический интерфейс, HTML-отчет, отчет в Grafana). Потестируем HTTP, JDBC, JMS. Рассмотрим, какими способами можно формировать данные для запроса: статика, генерация, извлечение из предыдущего запроса, чтение из файла.

Сделаем тест с использованием других приложений, допустим Gatling и Apache Benchmarking tool, поговорим в общем об утилитах тестирования производительности.</p> <p><a href="https://www.dropbox.com/s/wyzmoijjvkc2sbw/%D0%9F%D0%BE%D0%B4%D0%B3%D0%BE%D1%82%D0%BE%D0%B2%D0%BA%D0%B0_%D0%BA_%D0%B2%D0%BE%D1%80%D0%BA%D1%88%D0%BE%D0%BF%D1%83_%D0%A1%D1%82%D0%B0%D1%80%D1%82%D1%83%D0%B5%D0%BC_%D0%B2_%D1%82%D0%B5%D1%81%D1%82%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B8_%D0%BF%D1%80%D0%BE%D0%B8%D0%B7%D0%B2%D0%BE%D0%B4%D0%B8%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D0%BE%D1%81%D1%82%D0%B8%20%281%29.pdf?dl=0">Технические требования</a></p> <p>В случае возникновения вопросов по установке напишите организаторам воркшопа в телеграм: @Mahetovs @zausova.</p>
