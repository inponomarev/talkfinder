Вспомнить всех: Autofill Framework и Account Transfer API

Кирилл Борисов

Начав с азов использования Autofill Framework и Account Transfer API, разберем связанные с ними подводные камни и расскажем о нашем опыте внедрения этих инструментов.

Для разработчика мало что бывает страшнее, чем смена пользователем устройства: лишенный любимых настроек и утомленный повторным вводом логина/пароля, он может просто воспользоваться этим поводом и поставить что-то другое.

Но не всё так плохо — с выходом Android O в нашем арсенале появились новые средства, позволяющие и на новом устройстве порадовать пользователя привычным уютом:

  • Autofill Framework — автоматический ввод персональных данных пользователя;

  • Account Transfer API — возможность встроиться в системный процесс миграции данных между устройствами.

Из этого доклада вы сможете узнать азы использования этих инструментов и связанные с ними подводные камни, а также познакомиться с нашим опытом внедрения.
