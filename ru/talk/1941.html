Митап #20 JUG.MSK (15.12.2016 — 15.12.2016)

Разговор про Java 9

Иван Крылов

В 2017 ожидается выход новой версии Java – Java 9. Пришло время поговорить про то:

  • как скоро потребуется переезжать на новую версию;

  • чем являются модули и как переводить свои приложения на модульную архитектуру;

  • как взаимодействовать с немодульным кодом;

  • какие новые API появятся в Java 9.

Доклад является расширенной и значительно обновленной версией доклада на данную тему с конференций Jokerconf 2015 и Geekout 2016.
