Митап #24 JUG.ru (14.10.2013 — 14.10.2013)

Мастер-класс «Liferay 6.1 - строим корпоративный портал за 4 часа»

Евгений Цопа

Корпоративный портал обычно представляется со стороны как что-то ужасно сложное в сопровождении и трудное для изучения. На самом деле, это уже давно не так — портал на базе Liferay можно сконфигурировать и развернуть за считанные минуты, а чтобы научиться его администрировать и разрабатывать для него приложения, программисту, знакомому с Java будет достаточно нескольких часов. Тем не менее, чтобы начать работать с Liferay, нужно хотя бы в общих чертах понимать, как он работает, что из себя представляют плагины для него и владеть некоторыми шаманскими навыками, полезными при разработке этих плагинов. Эти вопросы и будут рассмотрены во время мастер-класса.

Тренинг будет состоять из трёх частей:

1. Знакомство с Liferay:

  • Что такое портал, общие понятия, сравнение Liferay с другими решениями для построения порталов.

  • Варианты установки портала, какой вариант для чего лучше подходит.

  • Конфигурация портала, основные конфигурационные файлы.

  • Настройка доступа к СУБД, конфигурация файлового репозитория.

  • Интерфейс портала, страницы, сайты, панель управления.

  • Сайты, "частные" и "публичные" страницы.*

*- Портлеты "из коробки", выбор шаблона размещения контента и темы оформления.

2. Liferay CMS:

  • Что такое "сетевой контент", как он размещается на страницах, какие портлеты существуют для этого.

  • Структуры данных и шаблоны сетевого контента — создание и использование на страницах портала.*

*- Разработка шаблонов, знакомство с Velocity Macros, использование предопределённых переменных.

3. Разработка плагинов для Liferay:

  • Основные виды плагинов, для чего они нужны, особенности их разработки и развёртывания.

  • Инструменты разработки плагинов для Liferay — обзор, особенности конфигурации.

  • Развёртывание плагинов на портале — с использованием IDE и "вручную".

  • Portlet MVC Framework — как быстро написать простой портлет под конкретную задачу.
