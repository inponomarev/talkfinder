Lightweight microservice collaboration using HTTP

Christian Horsdal

You’ll learn about different modes of collaboration between microservices, how they compare, which are preferable, and how to keep it all simple, along with how to use HTTP for the different types of collaboration and how to use Nancy to implement them on top of ASP.NET Core.

Комментарий Программного комитета:

Доклад от автора одной из самых крутых книг про микросервисы.

With a microservices architecture, system behavior gets spread out across many, many narrowly focused microservices. Much of the functionality of the system depends on those services collaborating. In this talk, we’ll explore different modes of collaboration between microservices, how they compare, which are preferable, and how to keep it all simple. Along the way, we’ll show how to use HTTP for the different types of collaboration and how to use Nancy to implement them on top of ASP.NET Core.
