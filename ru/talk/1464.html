<p>In this talk Victor will give a comprehensive overview of the the {fmt} library and the C++20 formatting library based on it. He will present a series of examples illustrating all major features (depending on the available time) such as:</p> <ul> <li>Format string syntax;</li> <li>Support for user-defined types;</li> <li>Formatting of chrono times and dates;</li> <li>Code bloat prevention via type erasure;</li> <li>Iterator support;</li> <li>I/O;</li> <li>Support for terminal colors and text styles;</li> <li>Unicode support;</li> <li>Compile-time checks;</li> <li>Fast integer and floating-point formatting;</li> <li>Locale support.</li> </ul> <p>The target audience is C++ developers interested in text processing, formatted I/O and new C++20 features. The main takeaway for the audience is understanding the powerful and efficient formatting facilities available in the modern C++ via open-source libraries such as {fmt} and soon in the standard libraries.</p>
