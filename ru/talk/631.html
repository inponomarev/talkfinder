Joker 2017 (03.11.2017 — 04.11.2017)

Тестирование lock-free алгоритмов, или Поиск иголки в стоге сена

Никита Коваль

Доклад о различных стратегиях и техниках для проверки на линеаризуемость, а также о тестировании своих алгоритмов на предоставляемом API.

Писать многопоточные программы сложно, однако тестировать их нисколько не проще. Продумать опасные многопоточные сценарии исполнения и написать для них тесты — трудная задача, поэтому зачастую ограничиваются простым набором стресс-тестов, которых не всегда хватает для выявления ошибки. Для решения этой задачи был разработан инструмент Lin-Check, автоматически проверяющий многопоточный Java-код на линеаризуемость. Первая часть доклада посвящена различным стратегиям и техникам для проверки на линеаризуемость. Во второй части обсудим предоставляемый API и как с помощью него тестировать свои алгоритмы.
