Heisenbug 2021 СПб (06.04.2021 — 09.04.2021)

Импорт и анализ автотестов с помощью библиотеки для PyTest в системе управления тестированием Test IT

Игорь Гросс, Александр Павлов

Клиентский кейс: партнер Test IT расскажет о процессе и результатах интеграции автотестов на Python на нескольких уровнях в один репортер, а также совмещение отчетности по автотестам с отчетностью по ручному прохождению в единой среде.
