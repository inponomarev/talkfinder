JPoint 2018 (06.04.2018 — 07.04.2018)

Погружение в Интернет Вещей с Java 9

Александр Белокрылов

Доклад даст вам инструментарий, позволяющий строить компактные приложения для сбора и препроцессинга потоковых данных на устройствах.

Комментарий Программного комитета:

Качественный доклад про IoT от человека, руководившего JavaME Embedded, Oracle IoT Cloud и делающего свой дистрибутив JDK для ARM. Александр занимался IoT, когда это ещё не называлось IoT.

Медленно, но верно Интернет вещей проникает в жизнь каждого. Java уверенно удерживает позиции не только в качестве технологии бэкенда, но и как платформа для гейтвея. Java 9 в дополнение к модульности принесла целый букет функциональности, бесценной для разработки IoT-решений. Этот доклад даст вам инструментарий, позволяющий строить компактные приложения для сбора и препроцессинга потоковых данных на устройствах.
