DotNext 2017 Мск (12.11.2017 — 13.11.2017)

Непрерывное развёртывание .NET Core приложений и инфраструктуры в Linux среде

Кирилл Ветчинкин

Продемонстрируем практический опыт организации процесса разработки и развёртывания .NET Core приложений на ОС Linux и настройки самих сред из кода. Также расскажем про плюсы и минусы, которые мы для себя извлекли от перехода на Linux.

В докладе будет продемонстрирован практический опыт организации процесса разработки и развёртывания .NET Core приложений на ОС Linux, а также настройка самих сред из кода. Для этого мы будем использовать связку таких средств, как: GitLab, TeamCity, Ansible, Octopus Deploy, Docker. Будет показана связь данных технологий для формирования единого конвейера. Также поговорим про плюсы и минусы, которые мы для себя извлекли от перехода на Linux.
