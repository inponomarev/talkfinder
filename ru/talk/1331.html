TechTrain 2019 (24.08.2019 — 25.08.2019)

AR — дополняем реальность

Данис Тазетдинов

Доклад о будущем дополненной реальности и о создании инструментов для навигации внутри помещений.

Рассказ об опыте создания инструментов для навигации внутри помещений с использованием технологий дополненной реальности, доступных в современных смартфонах от Apple.

На примере прототипа приложения для построения кратчайшего маршрута, обсудим вопрос, как работать с внутренней геометрией помещения.

Создавать эффективные AR-приложения возможно просто и быстро с помощью инструментария, доступного Apple-разработчику, «из коробки», такого как ARKit, SceneKit, UIKit, Swift, поставляемых с IDE Xcode. Также существуют практические результаты, которые позволили определить существующие ограничения технологий и дальнейшие идеи развития для целей Indoor Navigation.
