Восстанавливаем утраченную экспертизу по сервису

Анна Абрамова

Анна расскажет, какие есть подходы по восстановлению экспертизы по системам на примере вымышленной системы, написанной на Java и вспомогательных технологиях.

Достаточно часто, если вы идёте не в стартап или не во вновь созданное подразделение компании (команду, отдел, департамент), приходится сталкиваться с тем, что принято называть legacy (старый код, а иногда и технологии).

Вам сильно повезёт, если эта система будет описана хотя бы частично. А что делать, если нет? Если о ней не известно почти ничего?

Анна расскажет, какие есть подходы по восстановлению экспертизы по системам на примере вымышленной системы, написанной на Java и вспомогательных технологиях.
