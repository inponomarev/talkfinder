Apache Cayenne: альтернатива в мире Java ORM

Андрусь Адамчик

Рассказ об истории Apache Cayenne, немного о принципах Apache Foundation, Cayenne в действии, объяснение отличий с Hibernate/JPA.

Apache Cayenne — зрелый полнофункциональный Java ORM. На презентации будут обсуждаться подход Cayenne к персистентности объектов, общая философия и различия с JPA/Hibernate с фокусом на управление транзакциями, процесс маппинга, а также новые API-интерфейсы в последнем Cayenne 4.0.
