JPoint 2018 (06.04.2018 — 07.04.2018)

Graal, Value Types, Loom и прочие ништяки: что это и зачем оно нужно простому смертному

Олег Чирухин

Картина «поля боя» за свежую, современную и быстро развивающуюся Java: Graal/Truffle, Value Types, Amber, Loom, Shenandoah.

Коментарий Программного комитета:

Официально лучший автор Хабра (начиная с осени прошлого года), пишет на разные темы: Java, .NET, JavaScript и т.п. Отлично делает обзоры. Есть опыт продвижения нововведений в традиционно статичных государственных организациях.

В этом докладе мы кратко подведём итоги прошедшего года (выход Java 9 и Java 10, новая релизная политика) и обсудим основные направления развития OpenJDK: Graal/Truffle, Value Types, Amber, Loom, Shenandoah. Доклад будет полезен слушателям, желающим получить общую картину «поля боя» за свежую, современную и быстро развивающуюся Java.
