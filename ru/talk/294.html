Joker 2016 (14.10.2016 — 15.10.2016)

Контейнер не нужен: Современный Java Stack с Bootique.io

Андрусь Адамчик

Доклад о технологии Bootique.io. Андрусь расскажет об основных составляющих Bootique: модулях, конфигурации, командах и продемонстрирует как писать Java приложения в UNIX стиле, с интеграцией сложных модулей и полным интеграционным тестированием.

Будущее Java — в отказе от контейнера. Но без привычного контейнера программистам приходится организовывать свой собственный интеграционный стек, либо рассчитывать на существующие «opinionated» фреймворки. Проект Bootique предлагает альтернативный свежий подход. Его минималистичное ядро обеспечивает 3 основные вещи: POSIX совместимую подсистему запуска приложения, прозрачный механизм модуляризации и интеграции, и простой толковый движок конфигурации. В остальном Bootique старается не мешать программисту. Презентация продемонстрирует, как писать Java-приложения с Bootique в UNIX стиле, с интеграцией сложных модулей и полным интеграционным тестированием.
