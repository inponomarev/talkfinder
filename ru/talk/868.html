Building real world production-ready web APIs with .NET Core

Alex Thissen

You’ll learn how to build your evolving web APIs using .NET Core and the Microsoft platform and run it in a robust, hyper-scale production environment.

You might have started building your microservices-based web APIs using .NET Core. And you are ready to go into production. Did you think about hosting, scaling, monitoring, versioning, documentation, resiliency, and logging? Because going into production with just your business logic is not enough. In this session, you will learn how you have to build your evolving web APIs using .NET Core and the Microsoft platform and run it in a robust, hyper-scale production environment.
