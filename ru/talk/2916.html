TechTrain 2021 Spring (27.03.2021 — 27.03.2021)

Code with me: Behind the scenes

Кирилл Скрыган

Что такое Code With Me и какое значение эта функциональность имеет для рядовых разработчиков и для IT-организаций? Об этом расскажет один из создателей — Кирилл Скрыган.

Компания JetBrains выпускает этой весной прицнипиально новую функциональность для всех своих продуктов — Code With Me. Это платформа для коллаборативной работы — неважно, парное ли это программирование, обучение классов, воркшопы, или mob-программирование.

Какое значение это имеет для рядовых разработчиков и для IT-организаций? Как эта штука устроена внутри? Какая история и какое будущее у Code With Me? Обо всех этих вещах расскажет лид проекта, Кирилл Скрыган.
