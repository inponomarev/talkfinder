Mastering Auto Layout for iOS applications

Libranner Santos

We will cover some interesting topics about how Auto Layout works, and techniques we can use to debug issues related to it. Then we will see some of the best practices you can use while working with Auto Layout.

Auto Layout is a system of constrains based on the Cassowary algorithm. It helps iOS developers to create user interfaces that look good in all the screen sizes and devices.

In this talk we will cover some interesting topics about how Auto Layout works, and techniques we can use to debug issues related to it.

Then we will see some of the best practices you can use while working with Auto Layout.
