HolyJS 2018 СПб (19.05.2018 — 20.05.2018)

Bringing mobile web back to life

Imad Elyafi

We’ll share challenges, technologies and best practices we used in order to rebuild the whole mobile web experience for authenticated users in 11 weeks only!

Комментарий Программного комитета:

Честный опыт создания мобильного web-приложения, набор практик, не ограничивающийся PWA, с реальными метриками.

We rebuilt the whole mobile web experience for authenticated users in 11 weeks only! It looks and feels just like our native apps while maintaining the fast load times expected of mobile web. We used cutting edge technologies which led to a 40% increase in each of our core engagement metrics and up to 30% in time spent. But it was not all sunshine and rainbows! In this talk, we’ll share challenges, technologies and best practices that we used for the new Mobile Website.
