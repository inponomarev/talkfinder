Яндекс direvius

Занимается разработкой инфраструктурных сервисов и инструментов, связанных с тестированием производительности и мобильной разработкой. Развивает open source-проекты <a href="https://github.com/yandex/yandex-tank"; target="_blank">Яндекс.Танк</a>, <a href="https://github.com/yandex/pandora"; target="_blank">Pandora</a> и <a href="https://github.com/yandex-load/volta"; target="_blank">Volta</a>.

  • Учимся анализировать результаты нагрузочного тестирования (Heisenbug 2017 СПб, 04.06.2017 — 04.06.2017)

  • Тестирование телефонов с помощью Arduino (Heisenbug 2017 Мск, 08.12.2017 — 09.12.2017)

  • Круглый стол – Тестирование производительности и поиск узких мест (Heisenbug 2017 Мск, 08.12.2017 — 09.12.2017)

  • Pandora: нагрузочные тесты в виде кода (Heisenbug 2019 СПб, 17.05.2019 — 18.05.2019)
