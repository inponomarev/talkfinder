Andrey Karpov

PVS-Studio Code_Analysis

Andrey is a co-founder of PVS-Studio project and CTO at Program Verification Systems. He is Microsoft MVP in the nomination "Developer Technologies". He is an author of large number of articles dedicated to the code quality and error patterns that C++ developers make. He promotes methodology of static and dynamic code analysis.

  • Premature optimization is evil! All hail premature optimization! (C++ Russia 2019 SPb, 31.10.2019 — 01.11.2019)

  • What you should pay extra attention to at your library's code review (C++ Russia 2019 Msc, 19.04.2019 — 20.04.2019)

  • Статический анализ кода в 2020 году (C++ Russia 2020 Msc, 29.06.2020 — 02.07.2020)

  • Static analysis as hygiene code (DotNext 2015 Msk, 11.12.2015 — 11.12.2015)

  • What static analyzers can do while developers and testers can't (Heisenbug 2018 Msc, 06.12.2018 — 07.12.2018)
