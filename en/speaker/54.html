Gerrit Grunwald

Karakun AG hansolo_ HanSolo

Java Champion

Gerrit Grunwald is a software engineer with more than fifteen years of experience in software development. He is a true believer in open source and has participated in popular projects like JFXtras.org, as well as his own projects (TilesFX, Medusa, Enzo, SteelSeries Swing, SteelSeries Canvas).

Gerrit <a href="http://harmonic-code.org"; target="_blank">blogs</a> regularly, he is an active member of Java community, where he founded and leads the Java User Group Münster (Germany), is a JavaOne rockstar and Java Champion. He is a speaker at conferences and user groups internationally and writes for several magazines.
