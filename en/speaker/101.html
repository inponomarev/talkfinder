Ivan Osipov

Arrival Robotics ivan-osipov

Всю свою карьеру Иван разрабатывает и развивает бекенды веб приложений. Разработка серверсайда начала приносить ему еще больше радости, когда он встретил Kotlin. С момента этой встречи его жизнь изменилась на все 360°.

Последние полтора года в составе Arrival Robotics Иван с командой строят облачное решение, которое с одной стороны объединяет дизайнеров электротранспорта, а с другой гибкую роботическую фабрику. В перерывах между генерацией spring boot приложений Иван пишет телеграм-блог <a href="https://t.me/from_junior_to_senior"; target="_blank">@from_junior_to_senior</a>.

  • Kotlin DSL: theory and practice (Heisenbug 2018 SPb, 17.05.2018 — 18.05.2018)

  • Kotlin DSL: theory and practice (JPoint 2018, 06.04.2018 — 07.04.2018)

  • From Java to Kotlin sitting aft of a web application (JUG.MSK Meetup #43, 14.11.2019 — 14.11.2019)
