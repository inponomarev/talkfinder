Dmitry Patsura

LOWL ovrweb ovr

Developer with 10 years of experience, developed a lot of projected in open source, from a static analyzer — PHPSA, to the TypeScript compiler called StaticScript. Leading the most obscure podcast in the entire frontend — [UnderJS](https://underjs.ru "UnderJS")

Related open source: - [PHPSA](https://github.com/ovr/phpsa) — static analyzer on PHP for PHP; - [GHubber](http://github.com/ovr/ghubber) — mobile client for Github powered by React Native; - [StaticScript](https://github.com/ovr/StaticScript) — language on top of TypeScript as frontend, and LLVM framework as backend.

Dmitry also was a contributor to Zephir Language & Phalcon framework.

  • Compiler development for TypeScript on TypeScript based on LLVM (HolyJS 2019 Msc, 08.11.2019 — 09.11.2019)

  • Architecture of React Native for JS developer (HolyJS 2018 SPb, 19.05.2018 — 20.05.2018)

  • Microservice architecture (HolyJS 2018 Msc, 24.11.2018 — 25.11.2018)
