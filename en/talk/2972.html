Mobius 2021 SPb (13.04.2021 — 16.04.2021)

Make widget great again

Aleksandr Verestnikov

Alexander will talk about the widgets that Apple presented at WWDC 2020 and show how simple the WidgetKit framework is.

Alexander will talk about the widgets that Apple presented at WWDC 2020.

Many developers do not implement new technologies in their projects, because there is no time to study them, in the end, all this is put on hold, and then forgotten or is no longer relevant.

The purpose of the talk is to show how simple the WidgetKit framework is, and how quickly you can create a widget from scratch. Alexander will also talk about the main steps that need to be taken to create a widget and how to configure it.

This is not a complicated report, we will not go under the hood of widgets, but we will analyze everything you need to create them.

The key audience: developers who have not yet worked with widgets or are just starting to learn them.
