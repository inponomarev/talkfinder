DotNext 2018 Msc (22.11.2018 — 23.11.2018)

.NET Core 2.1: Intrinsics and other micro-optimizations

Egor Bogatov

This talk consists of interesting micro-optimizations, which were used in .NET Core by the developers and community members, and concentrates on newly minted API to the processor’s SIMD instructions, such as SSE and AVX.

This talk consists of interesting micro-optimizations, which were used in .NET Core by the developers and community members, and concentrates on newly minted API to the processor’s SIMD instructions, such as SSE and AVX. This API will help you feel smarter than JIT and vectorize your code manually, without using low-level languages.
