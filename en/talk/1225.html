Heisenbug 2019 SPb (17.05.2019 — 18.05.2019)

Vulnerability is a lucky bug

Artem Shishkin

Have you thought about what a vulnerability is? Would you ever think you are directly connected to this phenomenon? We’d like to discuss what types of bugs matter. Let’s see a real-life vulnerability example and how security researchers fight against it. Let’s raise our security awareness.

You might have heard some news lately talking about a newly discovered fancy vulnerability. Have you thought about what a vulnerability is? Would you ever think you are directly connected to this phenomenon? Because yes, we as IT guys are the potential source of significant real world problems. So what we’d like to discuss in this talk is what types of bugs matter. Let’s see a real-life vulnerability example and how security researchers fight against it. Let’s raise our security awareness.

This talk is addressed to developers and testers of all kinds. Let’s see how we can make a better world together.
