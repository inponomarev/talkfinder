DevOops 2017 (20.10.2017 — 20.10.2017)

Continuous Delivery in Windows environment

Vitaly Khabarov

We’ll tell you how we managed to automate the launch of test environments, with Windows as an application server, about the problems we faced and how we overcame them.

Is it possible to build a solid foundation for Continuous Delivery in an environment so unusual for DevOps? We’ll tell you how we managed to automate the launch of test environments, with Windows as an application server, about the problems we faced and how we overcame them.

The talk will be interesting for engineers going to implement DevOps practices in Windows environments, as well as for newcomers to DevOps, as some details of the Continuous Delivery process will be revealed.
