Mobius 2020 SPb (22.06.2020 — 25.06.2020)

Hacking and embedding your code in a foreign application

Murad Tataev

Hacking an iOS application using the example of a test Swift application and protection methods.

<p>You will learn how to hack iOS applications with next steps:</p> <ul> <li>what tools are used;</li> <li>how to bypass checks (with Jailbreak for example);</li> <li>how to code injections are performed;</li> <li>which protection methods are relevant and which brings the only inconvenience to the user. </li> </ul> <p>You will see the process of application hacking from downloading the app to signing and installing the modified app on the device. </p> <p>This talk will not address the topic of protection against attacks on the network layer. </p> <p>The session is ideal for Middle-Senior developers who want to learn (the dark side) hacking methods and use this knowledge to protect their applications.</p>
