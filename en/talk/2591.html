Heisenbug 2020 Msc (04.11.2020 — 07.11.2020)

Types of automated testing in IntelliJ IDEA

Yuri Artamonov

During this session we will see how to test IDE in JetBrains using automated tests and checks, what are the difficulties and which of this can be useful in other areas of software development.

Modern IDE is not just one product, it’s a whole platform for developing plugins and other IDEs with a huge amount of capability which can’t be tested manually by any team.

During this session we will see how to test IDE in JetBrains using automated tests and checks, what are the difficulties and which of this can be useful in other areas of software development.
