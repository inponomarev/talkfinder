C++ Russia 2019 SPb (31.10.2019 — 01.11.2019)

Automate it

Kirill Tikhonov

Kirill Tikhonov will tell how to configure the infrastructure around the project and provide the painless service update.

We will write a simple web-service from the client and server on C++. After that, the C++ part will end and set up the environment and infrastructure will start.

We will provide a build determinability and tests run.

Also, we will ease subsequent updating of dependent frameworks.

Automate static checks, code verification, tests run.

We will ensure the availability of the service, configure the infrastructure, balance the load and add automatic and manual scaling.

And in the end, we will configure continuous delivery so that code will be in production 5 minutes after request, while even invalid changes and programmer errors will not affect his work.
