Heisenbug 2020 Msc (04.11.2020 — 07.11.2020)

Workshop: Kick-start performance testing like a pro (part 2)

Sergey Mahetov

Workshop for testers who want to do performance testing but don’t know where to start.

<p>Workshop for testers who want to do performance testing but don't know where to start.</p> <p>We will create a test from scratch in Jmeter. We will make a basic query step by step, consider how to set a load profile, test metrics, provide reports (console, graphical interface, HTML report, a report in Grafana). We will also test HTTP, JDBC, JMS. We will consider in what ways the data for a request can be formed: statics, generation, retrieving from a previous request, reading from a file.

We will make a test using other applications, for example, Gatling and Apache Benchmarking tool, and talk in general about performance testing utilities.</p>
