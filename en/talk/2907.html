SnowOne 2021 (26.02.2021 — 27.02.2021)

Learning how to cook Spark (part 2)

Pasha Finkelstein

There are legends around saying that big data is hard. Well, truth is it’s really not always simple, but what is cool that when we’re using right tools difference with small data is not that big. On this workshop we’ll learn how to work with big data on example of small data — process, build charts and gather insights.

Together we’ll set up Big Data Tools and Zeppelin, will download couple of interesting datasets and will figure out how all this stuff works (on Scala).

Prerequisites: knowledge of any mainstream JVM language
