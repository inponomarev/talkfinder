Heisenbug 2020 Msc (04.11.2020 — 07.11.2020)

Test cases-as-code

Artem Eroshenko

During this talk, Artem will talk about the test-cases-as-code approach that is used to organize test automation.

<p>The world has an approach to organizing the product development process. It’s called Everything as Code. The main idea is to store the configuration of the development and delivery stages of the product right in the code. For example: build-as-code, documentation-as-code, delivery-as-code.</p> <p>And what about test cases? Manual work still the main approach in testing, even when we talk about automation. Often a test case is created by the hands and only then is automated.</p> <p>During this talk, Artem will talk about the test-cases-as-code approach that is used to organize test automation.</p>
