DotNext 2017 Msc (12.11.2017 — 13.11.2017)

.NET Core Networking stack and Performance

Karel Zikmund

We’ll dwell on the architecture of .NET Core networking stack focused on portability and performance, along with general .NET Core BCL performance goals.

Architecture of .NET Core networking stack focused on portability and performance. General .NET Core BCL performance goals. The talk will include:

  • Design goals of the new networking stack*

  • * Status of the implementation in .NET Core and early perf results

  • Performance goals and strategy for Networking stack and general BCL
