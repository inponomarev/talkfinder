Akka Streams for mere mortals

Vagif Abilov

Processing live streams of data as simple as writing LINQ expressions, without hand-written implementation of individual actors or Reactive Streams interfaces.

How can we just in a few lines of code display a live stream of Twitter messages, extending them with weather forecast in the locations of tweet authors? And how can we limit the number of requests to the weather service, so they won’t blacklist us? All of this you can see in the demo that will conclude the presentation.
