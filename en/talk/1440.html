C++ Russia 2020 Msc (29.06.2020 — 02.07.2020)

Understanding C++ coroutines by example

Pavel Novikov

Pavel will tell in detail how exactly C++ coroutines work, and what do we need to do to start using them easily and efficiently.

Coroutines are a powerful tool added to C++20. Though currently there are no means to conveniently work with coroutines out of the box. That’s why we need to learn to properly use this tool to get benefits and to do it efficiently. We’ll be doing just that.

We’ll figure out how to use coroutines from the ground up for work with aasynchronous tasks.
