Microservices: Single digit microseconds latency â€” friends and foes

Dmitry Pisklov

Common and less-known pitfalls and some ways to avoid those on a way to the ultra-low latency microservice architecture.

In this talk, we’ll discuss how to achieve ultra-low latencies in microservices. We’ll look into what affects latency: on hardware or software level, in an application, JVM, OS, chipset or CPU, and how to fight those factors to achieve the lowest latency. We’ll talk about thread affinity, CPU isolation, CPU cores affecting each other, as well as about shared memory and using technologies based on it as a microservice IPC.
