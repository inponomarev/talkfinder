Jetpack Compose for Desktop: Making UI is simple and pleasant

Nikolay Igotti

Nikolay will tell about Jetpack Compose technology which the JetBrains team ported from Android and adapted for use on desktop platforms.

The purpose of the talk is to tell about Jetpack Compose technology which the JetBrains team ported from Android and adapted for use on desktop platforms.

Content: a small description of the Jetpack Compose declarative framework, code examples, a more detailed focus on desktop development: what was added and why. A story about code reuse between Android and desktop. Plans, prospects.

What people will get: information about how easy and enjoyable it is to write UI applications on the desktop and reuse code from mobile applications.
