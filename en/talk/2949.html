JUnit 5 Parallel test execution. Concept and practice

Dmitry Tuchs

Parallel test execution — a talk about threads and tests on the JUnit 5 platform.

Let’s talk about what happens under the hood of the JVM when you enable parallel.enabled = true in JUnit5.

We will see what the threads are doing and answer the question — are your tests ready to run in parallel and what place does Java Concurrency take in autotests.
