JBreak 2016 (19.03.2016 — 19.03.2016)

Practical profiling of Java applications

Alexey Ragozin

Представьте, что перед вами Java приложение, которое «тормозит». В Java экосистеме недостатка в профайлерах, но одного инструмента мало, надо знать как им пользоваться.

В докладе будут освещены как общий подход к исследованию производительности Java приложений так и доступные инструменты профилирования.
