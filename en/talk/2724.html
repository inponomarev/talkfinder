HolyJS 2020 Msc (25.11.2020 — 28.11.2020)

WebAuthn: Theory and practice

Martin Splitt, Semyon Levenson

The talk about WebAuthn, what it is, and its specialties.

WebAuthn is some young but perspective technology. It provides not only improve security but also a more friendly UX to authentication.

In this talk you will listen about:

  • what is WebAuthn and what problems it solves;

  • what use cases has for WebAuthn;

  • how to add WebAuthn to your site and how this technology can help us to reduce the differences with native applications.
