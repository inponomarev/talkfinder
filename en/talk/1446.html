C++ Russia 2020 Msc (29.06.2020 — 02.07.2020)

DIY: Homemade Thread Pool

Denis Kormalev

The C++ Standard Library lacks a thread pool mechanism and it is not always convenient to use Boost, TBB or Qt for these purposes. This talk is about creating your own thread pool step by step without any dependencies except C++17.

<p>The C++ Standard Library lacks a thread pool mechanism and it is not always convenient to use Boost, TBB or Qt for these purposes. This talk is about creating your thread pool step by step without any dependencies except C++17.</p> <p>During the talk we will see how easy it is to create your minimal thread pool with decent performance and rich functionality. We will benchmark it and add optimizations based on those benchmarks. The talk also will tap into creating easy-to-use, yet flexible, the public interface for this thread pool.</p> <p>Talk is based on <a href="https://github.com/dkormalev/asynqro">Asynqro</a> library.</p>
