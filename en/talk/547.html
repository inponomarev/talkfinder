Offline Second

Alexey Bogachuk

How to add an offline mode to an already existing app.

Nowadays there are many tips on using «Offline First» principle in a greenfield project. But how to add an offline mode to an application that has already been released? What tactics and architectural approaches should be used? What technologies and libraries should one look for? What kinds of storage are needed for implementation of pull/push strategies?
