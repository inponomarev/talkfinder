Joker 2020 (25.11.2020 — 28.11.2020)

Working with in-memory data sharding with a flavor of Spring Data

Alexey Kuzin

Using the example of the Tarantool connector Alexey will show how to develop his implementation of a connector for a distributed in-memory data storage, what performance issues could arise, and how to solve them.

Spring Data is a powerful and widely used part of the Spring framework that provides abstractions over the work of different data stores. On the Internet, there is a lot of information about the use of prepared Spring Data modules, but for changes and development of your module, you have to dig deep into the sources.

Using the example of the Tarantool connector Alexey will show how to develop his implementation of a connector for a distributed in-memory data storage, what performance issues could arise, and how to solve them.

This talk will be of use for developers who are interested in the implementation details of the Spring framework and want to learn more about the details of working with a NoSQL database.
