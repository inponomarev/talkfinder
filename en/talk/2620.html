In the talk, Nikolay wants to show how to use webtau to test a game store app that has Web UI, REST API, GraphQL, and CLI interface.

<p>Mykola is one of the main contributors to webtau framework. Short for web test automation.</p> <p>The goal of the framework is to provide API, REPL and standalone CLI runner to test across multiple layers such as HTTP, Web UI, DB, CLI with a consistent API, reach reporting, and extensibility.</p> <p>The framework has reach DSL to help you write things concisely and robustly.</p> <p>In the talk, Mykola wants to show how to use webtau to test a game store app that has Web UI, REST API, GraphQL, and CLI interface.

Webtau also has a new (recently released) Persona concept to simplify testing authorization scenarios. It can also be used to control more than one browser at a time to test things like web sockets.</p> <p>Speaker will also show how to use tests to capture data for documentation purposes and generate documentation for an application under test.</p>
