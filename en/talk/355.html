DotNext 2016 Msc (09.12.2016 — 09.12.2016)

F# for the glory of Data Science

Roman Nevolin

Usage of the functional power of F # language for data analysis and processing tasks.

What are Data Scientist need to be Absolutely Happy? Of course, data and the bigger, the better. And also a good tool for processing and analysis data, the possibility of interactive processing and many different visualizations.

For .NET such a tool is F#. In this talk, Roman along with an audience will look at language features that make it so attractive for Data Science, consider the most useful libraries for this purposes, and will look at some real-world use cases.

Basic F# knowledge can be useful, but not required.
