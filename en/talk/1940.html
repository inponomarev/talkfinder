JUG.MSK Meetup #19 (14.11.2016 — 14.11.2016)

Harmful MVC architecture

Yegor Bugayenko

MVC (Model View Controller) — супер популярная архитектура для веб и GUI приложений в Java мире. Однако, зачастую возникает вопрос, так ли она хороша в ее типичном исполнении?

Мы рассмотрим несколько альтернативных решений и докладчик предложит свой вариант. На практическом примере небольшого Java приложения разберем как именно MVC может быть полезен, а как вреден.
