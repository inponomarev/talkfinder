Heisenbug 2019 SPb (17.05.2019 — 18.05.2019)

Accelerating Apache JMeter

Vyacheslav Smirnov

In the projects requiring high load, performance of a load script becomes a crucial question. And so we’ve tested performance of the performance testing tool.

Apache JMeter is a popular performance testing tool with a lot of components and capabilities. With it, one and the same operation can be performed in several possible ways.

In the projects requiring high load, performance of a load script becomes a crucial question. And it would be great to have performance rating of Apache JMeter components and approaches to script writing.

Using Java applications profiling means, such as Java Flight Recorder, jVisualVM, SJK, having access to the tool source code, having written synthetic tests, and having taken some real-life examples, we’ve tested performance of the performance testing tool.

This talk could be of interest for performance testing engineers using Apache JMeter, juniors and seniors alike, and for developers using JVM/JDK for work and dealing with code profiling and optimization.
