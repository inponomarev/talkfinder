Workshop: How to start your automation project from scratch. Sequel

Andrei Solntsev

In the first workshop on Heisenbug 2020, we learned how to start an automation project from the scratch.

We created a new project, built it with Maven/Gradle, wrote basic Selenide code, added Allure and Selenoid, and executed tests with github actions.

Now we will dive deeper into test automation. We will learn next, more advanced techniques.

  1. Write advanced tests for TODO MVC.

  2. Write tests for "enterprise" websites with non-trivial locators.

  3. Try Record-and-Play for Selenide.

  4. Learn to work with proxy in Selenide.
