Mobius 2021 SPb (13.04.2021 — 16.04.2021)

The functionality of shooting spherical panoramas in the mobile application from A to Prod. Developing, implementing and using

Gennady Vasilkov

The talk is about the development and implementation of their own library for shooting 360° panoramas.

Gennady will share a recipe for developing functionality for creating and displaying 360° panoramas on a mobile device. He’ll tell you how DomClick came to the need for such a solution in their service, where to start, how to continue and how to successfully complete the development and implementation on mobile clients.

It will be useful primarily for those who need to use rather complex functionality on a mobile device that can be transferred to a cross-platform library.

Ingredients:

  • Stiching — 1 pc. Self-developed C ++ cross-platform library;

  • Mobile applications — 2 pcs. iOS and Android;

  • CI/CD — 1 pc. Jenkins.

Shake but do not mix. Add developers.
