The inner structure of a webpack bundle

Alexey Ivanov

How does webpack create bundles, transform code, choose which files to add to build, connect chunks? Knowing all this you can debug your code easier and control bundle size.

webpack is a de-facto standard bundler for large frontend projects. Lots of developers use it, but mostly like a black box: you place your files in some place, write some magical words in config and then magic happens. Until it doesnâ€™t.

The talk will explain how this magic works, how a webpack bundle looks from inside, how different settings can change output and lead to unexpected side effects. And also how you can debug and optimize it.
