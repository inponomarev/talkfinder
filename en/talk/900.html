Configuration testing for Java developers: a case study

Ruslan Cheremin

We’ll discuss how we write Java unit-tests for configuration, how we benefit from them, pitfalls and workarounds.

Andrew Satarin has given a talk on recent Heisenbug about how to cover configuration with tests.
