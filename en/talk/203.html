Heisenbug 2019 Msc (05.12.2019 — 06.12.2019)

Automation of automation department

Irina Rubchenko

During this session, Irina will try to figure out why no one likes to write test cases and how to overcome this hostility through process automation.

Are you familiar with the situation when you need to write 1000+ test cases and E2E tests to automate web application?

Regression takes more than 3 days and there is no time to write test cases. Imagine that there is a team that managed to get rid of this pain.

Irina will tell how to fully automate the process of writing test cases and E2E tests for web application with highly loaded interface.
